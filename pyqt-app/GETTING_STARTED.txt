DNA PATTERN MATCHER - QUICK START
==================================

WHAT IS THIS?
=============
A DNA sequence pattern matcher using Finite Automata (DFA).
For Theory of Automata course project.

FEATURES:
- Pattern matching with O(n) time complexity
- DFA state visualization
- FASTA file support
- AI pattern extraction (optional, with Google Gemini)
- Real-time statistics and charts

HOW TO RUN:
===========

1. Activate virtual environment:
   source .venv/bin/activate

2. Run the app:
   streamlit run app_web.py

3. Opens at: http://localhost:8501

QUICK TEST (No API Key Needed):
===============================

Pattern: ATCG

TO USE AI FEATURE (Optional):
=============================

1. Get API key: https://aistudio.google.com/app/apikey
2. Set: export GOOGLE_API_KEY="your-key"
3. Run: streamlit run app_web.py
4. Type: "find TATA box" in AI field
5. Pattern auto-extracts!

WHAT YOU CAN DO:

✓ View statistics and charts
